Aniyomi Guide

The Ultimate Aniyomi Setup

Everything you need to know about setting up Aniyomi with Extension Repositories and Torrent Addons for a perfect anime and manga experience.

Step 1: Downloading Aniyomi

Aniyomi is not available on the Google Play Store, so you'll need to download it from the official website or use a fork.

Note: The Stable version is recommended for most users to avoid bugs, while the Preview version contains the latest experimental features.
Download APK

Once downloaded, open the APK file to install Aniyomi. You might need to allow installation from unknown sources in your Android settings.

Step 2: Initial Setup

When you first open Aniyomi, follow the onboarding process carefully to set up your storage correctly.

  • Choose your preferred theme and click Next.
  • Select a storage folder. It is recommended to create a new folder called Aniyomi in your internal storage or SD card.
  • Grant the application permission to access your storage so it can save manga chapters and anime episodes.
  • If you use tracking services (like MyAnimeList or AniList), go to More > Settings > Tracking to sign in.

Step 3: Adding Extensions

Aniyomi relies on community-maintained extensions to fetch anime and manga. You must add these repositories to Aniyomi.

Adding Repositories Manually:

  1. Go to More > Settings > Browse.
  2. For Anime, click on Anime extension repos. For Manga, click Manga extension repos.
  3. Click + Add and paste the repository URLs:
Kohi Den (Anime): https://raw.githubusercontent.com/kohi-den/extensions/main/index.min.json
Secozzi (Anime): https://raw.githubusercontent.com/Secozzi/aniyomi-extensions/refs/heads/repo/index.min.json
Yuzono (Anime): https://raw.githubusercontent.com/yuzono/anime-repo/repo/index.min.json
Keiyoushi (Manga): https://raw.githubusercontent.com/keiyoushi/extensions/repo/index.min.json

Step 4: Torrentio Anime (Optional)

If you want to stream torrents directly (usually aided by a Debrid service), you can install the Torrentio Anime extension.

Configure Torrentio Anime

  1. Go to Browse > Anime Extensions and search for Torrentio Anime. Install it.
  2. Click the settings icon next to the extension, then click settings next to Multi.
  3. Select your Debrid Provider (e.g., Real Debrid, AllDebrid, TorBox).
  4. Enter your API Token from your Debrid provider.

You're All Set! 🎉

Search for any anime or manga in the Browse tab, add to your Library, and enjoy!
